# Ejercicio 1

Crear una página web que cumpla las siguientes condiciones:

> No importa sobre que trata la página, sean creativas. Tampoco importa que tan lindo se vea, solo importa el contenido.

- Debe existir por lo menos un elemento ``h1``
- Debe existir por lo menos un elemento ``h2``
- Deben existir por lo menos tres elementos ``p``
- Deben existir por lo menos dos elementos ``img``
- Deben existir por lo menos dos elementos ``button``
- Debe existir por lo menos una lista ``ol`` o ``ul`` con algunos ``li``
- Al ingresar a la página, debe aparecer un cartel de alerta con algún mensaje
- El color de fondo de la página debe tener algún color
- El color de la fuente de la página debe tener algún color

EXTRA: (Si no pueden hacer este, no hay problema. Pero estaría bueno que si.)

> Si quieren intentar hacer el EXTRA, recomiendo dejarlo para el final.

- Todo el contenido de la página debe estar centrado

AYUDA:

La regla para centrar uno o más elementos se utiliza de la siguiente manera:

```css
text-align: center;
```
